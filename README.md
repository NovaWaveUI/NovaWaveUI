# NovaWaveUI

NovaWaveUI is a rebrand from MotionWindUI! It is a UI library built for React, TailwindCSS, and Framer Motion. It is a collection of components that are easy to use and customize. It is built with TailwindCSS and Framer Motion to provide a smooth and responsive experience. It is also built with React to provide a simple and easy way to use the components.
